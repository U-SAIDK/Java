**  MAP **

1) The Map interface is a part of the Java Collections Framework and is used to store key-value pairs
2) Each key must be unique, but values can be duplicated.
3) A Map is useful when you want to associate a key (like a name or ID) with a value (like an age or description).


Common classes that implement Map:

HashMap - fast and unordered
TreeMap - sorted by key
LinkedHashMap - ordered by insertion



Common Map Methods

Method	        Description
put()	        Adds or updates a key-value pair
get()	        Returns the value for a given key
remove()	    Removes the key and its value
containsKey()	Checks if the map contains the key
keySet()	    Returns a set of all keys
