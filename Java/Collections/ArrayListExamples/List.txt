Java List Interface :-

1) The List interface is part of the Java Collections Framework and represents an ordered collection of elements.
2) You can access elements by their index, add duplicates, and maintain the insertion order.
3) Since List is an interface , you cannot create a List object directly.
   Instead, you use a class that implements the List interface, such as:
  . ArrayList - like a resizable array with fast random access
  . LinkedList - List with fast insert and remove operations

4.1) How the ArrayList works
   The ArrayList class has a regular array inside it. When an element is added, it is placed into the array.
   If the array is not big enough, a new, larger array is created to replace the old one and the old one is removed.

4.2 ) How the LinkedList works
The LinkedList stores its elements in "containers." The list has a link to the first container and each container has a link to the next container in the list.
To add an element to the list, the element is placed into a new container and that container is linked to one of the other containers in the list.

5 ) List Methods
    add()	    Adds an element to the end of the list
    get()	    Returns the element at the specified position
    set()	    Replaces the element at the specified position
    remove()	Removes the element at the specified position
    size()	    Returns the number of elements in the list
